MY MOVIES_API

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

TABLE

Request URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user '/movies' GET none A JSON object holding data about all movies
Return data about a single movie by title to the user '/movies/:Title' GET none A JSON object holding data about the movies
Return a list of ALL users to the user '/users' GET none A JSON object holding data about all users, structured like: {"_id":{"$oid":"6201a3dfa3b7ac67dba1ac3e"},"Username":"Bai Yu","Password":"abcbbc123","Email":"123edc@gmx.de","Birthday":{"$date":"1995-12-11T00:00:00.000Z"},"FavoriteMovies":[{"$oid":"6201b6cdc566b05a26336db6"},{"$oid":"6201af4ac566b05a26336dae"}]}
Return data about a single user by Title to the user '/movies/:Title' GET none A JSON object holding data about the movie
Return data about a director by name '/directors/:directorName' GET none A JSON object holding data about the director, structured like: {"Director": { "Name": "Jonathan Demme", "Bio": "Robert Jonathan Demme was an American director, producer, and screenwriter.", "Birth": "1944", "Death": "2017" } }
Return data about a genre description by genre name. '/genre/:genreName' GET none A JSON object holding the description of the genre
Allow new users to register '/users' POST A JSON object holding data about a user to add, structured like: "Username":"Susanna Hansent","Password":"susannalove","Email":"susanna@gmx.de","Birthday":{"$date":"1948-03-04"},"FavoriteMovies":[ A JSON object holding data about the user that has been just added, structured like: {"_id":{"$oid":"6201a751a3b7ac67dba1ac41"},"Username":"Susanna Hansent","Password":"susannalove","Email":"susanna@gmx.de","Birthday":{"$date":"1948-03-04T00:00:00.000Z"}}
Allow users to update their user information '/users/:Username' PUT A JSON object holding data about a user to change data, structured like: { "Username": "testUser", "Password": "1234567", "Email": "testUser@yahoo.com", "Birthday": "1950-04-25", } A JSON object holding data about users updated data, structured like: { "_id": "61fa65249526ff993d4b0ab4", "Username": "testUser", "Password": "changedPassword", "Email": "testUser@yahoo.com", "Birthday": "1950-04-25T00:00:00.000Z", "FavoriteMovies": [] }
Allow users to add a movie to their list of favorites '/users/:Username/movies/:MovieId' POST none A message indicating whether the movie was successfully added
Allow users to remove a movie from their list of favorites '/users/:Username/movies/:MovieId' DELETE none A message indicating whether the user was successfully removed
Allow existing user to deregister '/users/:Username' DELETE Return a text confirming that the user was successfully deleted.
