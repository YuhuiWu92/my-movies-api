MY MOVIES_API

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

TABLE

Request URL HTTP Method Response
Return a list of ALL movies to the user '/movies' GET Returns a JSON list of movies.
Return data about a single movie by title to the user '/movies/:title' GET Returns a JSON data about a specific movie.
Return data about a director by name "/movies/directors/:directorName" GET Returns a JSON data about the director.
Return data about a genre description by genre name. "/movies/genre/:genreName" GET Returns a JSON data with a description of that genre.
Allow new users to register /users/:newUser POST Will add new user to JSON object.
Allow users to update their user name "/users/:id" PUT Returns the data fields that was updated by the user.
Allow users to add a movie to their list of favorites "/users/:id/:movieTitle" POST Returns a text confirming which movie was added successfully.
Allow users to remove a movie from their list of favorites "/users/:name/:favouriteMovies" DELETE Returns a text confirming which movie was removed successfully.
Allow existing user to deregister "/users/:id" DELETE Return a text confirming that the user was deleted.
